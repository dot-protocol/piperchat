# code of conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) verbatim. The full text is published and maintained at the link above and is licensed under [CC BY 4.0](https://creativecommons.org/licenses/by/4.0/).

By participating in this project — opening issues, sending pull requests, commenting, or otherwise contributing — you agree to abide by it.

## scope

The Code of Conduct applies within all project spaces (this repository, its issue tracker, its discussion forums, code review threads) and when an individual is officially representing the project in public spaces.

## reporting

To report an incident, open a private email thread with the maintainers. If the project does not yet list a maintainer email, open a confidential GitHub Security Advisory on this repository instead. All complaints will be reviewed and investigated promptly and fairly.

All reporters are entitled to confidentiality regarding the report. Maintainers who fail to respect that confidentiality will face temporary or permanent consequences as judged by the rest of the maintainer team.
